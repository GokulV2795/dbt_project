# dbt_project
Initialize a dbt project for a retail analytics pipeline with bronze-layer staging models over raw source tables.

New Features:

Add initial dbt_project.yml configuration with model paths, environment settings, and bronze/silver/gold materializations.
Introduce bronze-layer staging views for customers, orders, order items, products, stores, employees, promotions, and returns.
Define retail_raw source configuration pointing to the analytics database and raw tables.
